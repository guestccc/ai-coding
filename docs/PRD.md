# 英语学习训练移动端Web项目 - 产品需求文档

## 1. 产品概述

### 1.1 产品愿景
打造一个专注于英语学习的移动端Web应用，通过智能化的训练方式和个性化的学习路径，帮助用户高效提升英语听说读写能力。

### 1.2 产品定位
- **平台类型**: 移动端优先的响应式Web应用
- **目标市场**: 中国英语学习者（学生、职场人士、语言爱好者）
- **核心价值**: 随时随地学习、个性化推荐、科学训练方法

## 2. 目标用户

### 2.1 主要用户群体
1. **学生群体** (60%)
   - 年龄: 12-25岁
   - 需求: 应试准备、日常学习、兴趣培养
   - 使用场景: 课后复习、考前冲刺

2. **职场人士** (30%)
   - 年龄: 25-40岁  
   - 需求: 商务英语、职业发展、海外交流
   - 使用场景: 通勤时间、工作间隙

3. **语言爱好者** (10%)
   - 年龄: 不限
   - 需求: 兴趣学习、文化了解
   - 使用场景: 休闲时间

## 3. 功能需求

### 3.1 P0 - 核心功能（MVP）

#### 3.1.1 用户管理
- **用户注册/登录**: 支持手机号、邮箱注册，第三方登录（微信、QQ）
- **个人资料**: 基本信息、学习目标设置、水平评估
- **学习进度跟踪**: 每日学习时长、完成课程数、能力提升曲线

#### 3.1.2 词汇学习
- **智能词库**: 按主题分类（日常、商务、考试等）
- **记忆算法**: 基于艾宾浩斯遗忘曲线的复习提醒
- **多种学习模式**: 卡片学习、选择题、拼写练习

#### 3.1.3 听力训练
- **分级听力材料**: 从初级到高级的音频内容
- **实时语音识别**: 跟读评分功能
- **听力理解练习**: 选择题、填空题

#### 3.1.4 口语练习
- **AI口语评测**: 实时发音评分和纠正建议
- **情景对话**: 模拟真实对话场景
- **跟读训练**: 句子跟读和对比

### 3.2 P1 - 进阶功能

#### 3.2.1 阅读训练
- **分级阅读材料**: 新闻、故事、文章等
- **阅读理解题**: 主旨题、细节题、推理题
- **生词标注**: 点击查词并加入生词本

#### 3.2.2 写作练习
- **话题写作**: 给定话题进行写作练习
- **AI批改**: 语法检查、表达建议
- **范文参考**: 优秀范文展示

#### 3.2.3 个性化推荐
- **学习路径推荐**: 基于用户水平和目标推荐课程
- **薄弱环节强化**: 针对错误率高的内容重点训练
- **每日计划**: 智能生成每日学习任务

### 3.3 P2 - 扩展功能

#### 3.3.1 社区功能
- **学习小组**: 创建或加入学习小组
- **成就系统**: 徽章、等级、排行榜
- **经验分享**: 用户学习心得交流

#### 3.3.2 离线学习
- **内容下载**: 支持离线学习词汇和听力材料
- **进度同步**: 联网后自动同步学习数据

#### 3.3.3 多端同步
- **Web/移动端同步**: 学习进度跨设备同步
- **数据备份**: 云端存储学习记录

## 4. 非功能性需求

### 4.1 性能需求
- **页面加载时间**: 首屏加载 < 2秒
- **响应时间**: 用户操作响应 < 200ms
- **并发支持**: 支持1000+用户同时在线

### 4.2 安全需求
- **数据加密**: 用户数据HTTPS传输
- **隐私保护**: 符合GDPR和中国网络安全法
- **防作弊**: 学习数据真实性验证

### 4.3 可用性需求
- **响应式设计**: 适配各种移动设备屏幕
- **无障碍访问**: 支持屏幕阅读器等辅助功能
- **多语言支持**: 中英文界面切换

### 4.4 可靠性需求
- **服务可用性**: 99.9% uptime
- **数据持久性**: 数据丢失率 < 0.01%
- **错误恢复**: 自动错误检测和恢复机制

## 5. 用户故事

### 5.1 学生用户故事
**作为学生，我想要**进行词汇记忆训练，**以便**提高我的词汇量应对考试。
- **验收标准**: 
  - 可以按主题选择词库
  - 支持多种记忆模式（卡片、选择、拼写）
  - 系统根据遗忘曲线提醒复习
  - 学习进度可视化展示

### 5.2 职场用户故事  
**作为职场人士，我想要**练习商务英语口语，**以便**在工作会议中更自信地表达。
- **验收标准**:
  - 提供商务场景对话练习
  - AI实时发音评测和纠正
  - 可以录制和回放自己的发音
  - 获得具体的改进建议

### 5.3 初级用户故事
**作为英语初学者，我想要**从基础开始学习，**以便**建立良好的学习 foundation。
- **验收标准**:
  - 提供水平测试确定起始点
  - 推荐适合初学者的课程内容
  - 学习难度循序渐进
  - 有详细的新手引导

## 7. 技术约束

### 7.1 前端技术栈
- **框架**: React 18 + TypeScript
- **UI库**: Ant Design Mobile
- **状态管理**: Redux Toolkit
- **构建工具**: Vite
- **包管理器**: pnpm

### 7.2 后端技术栈
- **语言**: Node.js
- **框架**: NestJS
- **数据库**: PostgreSQL + Redis
- **API**: RESTful + GraphQL
- **部署**: Docker + Kubernetes

### 7.3 第三方服务
- **语音识别**: 科大讯飞/百度语音
- **AI评测**: 自有算法或第三方API
- **推送服务**: 极光推送
- **统计分析**: Google Analytics

### 7.4 兼容性要求
- **浏览器**: Chrome, Safari, Firefox, Edge
- **移动端**: iOS Safari, Android Chrome
- **屏幕尺寸**: 适配320px-768px宽度
